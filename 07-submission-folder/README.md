Usage:
Download the data from the link:
https://huggingface.co/datasets/imone/D4RL/tree/main
Then run ```01-preprocess.py```
Then run ```02-network-sweep.py```
The second file accepts arguments --datadir and --resultsdir specifying respectively the directory the processed data is specified in and where the results must be stored.
Then run any of the plot files to create a figure of the results.
